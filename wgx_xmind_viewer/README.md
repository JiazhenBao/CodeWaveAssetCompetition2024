# 依赖库名称

## 依赖库设计

这个依赖库旨在提供一个xmind预览组件，用于在低代码开发中快速预览xmind文件。

## 主要特性

### 组件一：xmind预览组件

这个组件用于在低代码开发中预览xmind文件。它提供了一个简单的界面，允许用户只需要提供xmind文件地址并在界面上显示xmind文件的内容，并且支持缩放以及全屏观看。
